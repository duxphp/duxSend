# 简介

DuxSend 是一款支持多种平台接口的短信、邮件等推送类

# 支持平台

- 本地邮件
- 阿里云邮件
- 阿里云短信
- 小米推送
- 云片短信

  更多加入中...

# 环境支持

- 语言版本：PHP 7.1+
